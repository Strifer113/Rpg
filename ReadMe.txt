
# Text-Based Dungeon Crawler Game (v1.0.0)

## Overview

This is a classic text-based dungeon crawler game implemented in Python. Players embark on a perilous journey through a mysterious dungeon, battling monsters, collecting treasure, and uncovering secrets. The game is designed to be simple yet engaging, providing a nostalgic experience for fans of text-based adventures.

## Features

*   **Exploration:** Navigate through interconnected rooms, each with unique descriptions.
*   **Combat:**  Engage in turn-based battles against various monsters.
*   **Inventory Management:** Collect and equip weapons, armor, and other items.
*   **Hidden Items:** Search rooms to discover hidden treasures.
*   **Random Encounters:** Experience unexpected events and enemy encounters.

## Files

*   `main_game.py`: Main game loop, player interaction, combat logic, inventory management.
*   `game_data.py`: Classes for `Weapon`, `Armor`, and `Enemy`, and data for items and monsters.
*   `game_map.py`: Dungeon map structure with interconnected rooms and descriptions.

## How to Play

1.  Ensure you have Python installed.
2.  Place all files in the same directory.
3.  Run: `python main_game.py`
4.  Follow on-screen prompts to create your character, move, search, battle, and make choices.
